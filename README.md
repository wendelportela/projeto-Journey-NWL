# 🚀 Journey - NLW

Projeto desenvolvido durante o evento **Next Level Week (NLW)** da Rocketseat.  
A aplicação foi criada com o objetivo de praticar conceitos de desenvolvimento web moderno e construção de interfaces interativas.

## 📌 Sobre o projeto

O **Journey** é uma aplicação que permite organizar experiências e atividades de viagem, ajudando usuários a planejar seus destinos, registrar atividades e acompanhar sua jornada.

Este projeto foi desenvolvido com foco em aprendizado prático de tecnologias modernas utilizadas no desenvolvimento web.

## 🛠️ Tecnologias utilizadas

As principais tecnologias utilizadas no projeto:

- HTML
- CSS
- JavaScript
- React
- Node.js
- Git e GitHub
